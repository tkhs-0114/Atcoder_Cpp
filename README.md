# Atcoder_Cpp
Docker上のAtcoderで使用するC++の環境

### 使用コマンド
- 起動
```console
docker compose up -d
docker compose exec gcc bash
```
- 実行
input.txtの内容を入力とする
```bash
sh run
```
- 終了
```bash
exit
```
```console
docker compose down
```
